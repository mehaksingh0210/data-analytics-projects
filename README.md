# Intelligent-Excuse-Generator
AI-based excuse generator using Flask
An AI-inspired web application that generates context-based excuses with a believability score.

## Features
- Generate excuses based on:
  - Context (College/Office)
  - Reason (Health, Technical, Emergency)
  - Urgency level
- Believability scoring system
- Simple and clean UI

## Tech Stack
- Python (Flask)
- HTML, CSS
- JavaScript (basic)
